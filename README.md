 # STUDENT REGISTATION SYSTEM 
 -----------------------------
 This project is created by me as part of Javascript assignment.

 It is a simple web application that stores student information using a registration form and shows its in a table below .

The Registration system allow us to add, edit, delete, search and store student information.
\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\\
#  FEATURES 
-------------
1. Adding new student
2. Edit any student informatiom
3. Delete a student information record
4. Searching bar  
5. Auto serial number update
6. total registered students 
7.  Local storage
8.  works on mobile, tablet and laptop

# HOW TO USE
-------------------------
1. Download the student registation system folder
2. Open the folder
3. Double-click on index.html file
4. File will open in the browser
5. Fill all the fields in the registation form
6. Click Register button 
7. After completing registation students  details will apper in table
8. In table you show click Edit button  if you want to change data
9. In table you show click Delete button if you want to remove a student
10. Write anything in the search bar to filter students data
11. Local Storage keeps the data safe even after refresh page
